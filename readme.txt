Chat with operator button for your site using ChatUP hosted service

== Description ==
ChatUP is an hosted service for operator support provisioning to websites' visitors.
The operator logs in the ChatUP and waits visitors chat requests. The chat is fullly web-based.

= Features =
* Simply configure your campaign ID
* Chat button widget
* Italian and English translation

== Installation ==
1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your wp-content/plugin directory
2. Activate the plugin through the WordPress admin panel
3. Register on http://chatup.it and obtain your campaign ID
4. Insert the campiagn ID provided on the ChatUP settings page of the admin panel
5. Let ChatUP site verify the script on your site (to prove the ownership)
6. Access ChatUP operator panel and go online
7. You're ready to visitor's chat requests

== Frequently Asked Questions ==
= Is the service free? =
There is a trial period free, then for high traffic sites there is an annual fee. The restriction is based on the number of exposures of the "Operator Available" button.
We estimated that the restricion allows the majority of personal websites to continue to use the free version.
